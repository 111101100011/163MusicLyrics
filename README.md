网易云音乐歌词获取

## 使用方法：

1.前往网易云音乐官网:http://www.music.163.com

2.选择要获取的歌曲，进入播放页

3.获取页面的url链接中id后面的数值。例如歌曲凉凉播放页url为：http://music.163.com/#/song?id=479408220 ，则数值为479408220。

4.打开应用，输入获取到的数值，点击搜索即可。
